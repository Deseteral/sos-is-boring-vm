=== Summary ====================================================================
* 32-bit architecture
* numbers are stored in two's complement 
* big endian
* registers:
  * 16 general use registers: R0 .. R15
  * loop counter: LC
  * stack pointer: SP
  * program counter: PC

In this document, anything within [brackets] is shorthand for "the value of the
RAM at the location of the value inside the brackets". For example, SP register
is "stack pointer", but [SP] means the value of the RAM at the location the
stack pointer is pointing to.

=== Instruction length =========================================================

Instructions are always 32 bits long.

 Opcode: 8 bits
Value A: 8 bits
Value B: 8 bits
Value C: 8 bits
----------------
  Total: 32 bits

OOAABBCC

=== Instruction set ============================================================

--- Opcode table ---------------------------------------------------------------
value  | opcode     | description
-------+------------+-----------------------------------------------------------
0x00   | NOP        | no operation
0x01   | HCF        | halt
- - - -+- - - - - - + - - - - - - - - - - - - - - - - - - - - - - - - - - - - -
0x02   | MOV  a b   | sets a to value of b when b is number,
       |            |   sets a to b otherwise
0x03   | ADD  a b   | sets a to a + b
0x04   | SUB  a b   | sets a to a - b
0x05   | MUL  a b   | sets a to a * b (signed only)
0x06   | DIV  a b   | sets a to a / b (signed only)
0x07   | MOD  a b   | sets a to a % b (signed only)
0x08   | SAL  a b   | sets a to a << b
0x09   | SAR  a b   | sets a to a >> b (sign bit preserved)
0x0A   | INC  a     | sets a to a + 1
0x0B   | DEC  a     | sets a to a - 1
0x0C   | SHL  a b   | sets a to a << b
0x0D   | SHR  a b   | sets a to a >> b
0x0E   | AND  a b   | sets a to a & b
0x0F   | OR   a b   | sets a to a | b
0x10   | XOR  a b   | sets a to a ^ b
0x11   | NOT  a     | negates a, and stores the result in a
- - - -+- - - - - - + - - - - - - - - - - - - - - - - - - - - - - - - - - - - -
0x12   | PUSH a     | push a to stack
0x13   | POP  a     | pop value from stack and set a to it
- - - -+- - - - - - + - - - - - - - - - - - - - - - - - - - - - - - - - - - - -
0x14   | JMP  a     | sets PC the memory address of label a
0x15   | JMR  a     | adds value of a to PC
0x16   | CMP  a b   | compares a with b and sets flags
0x17   | JZ   addr  | executes "JMP addr" if ZERO flag is set
0x18   | JNZ  addr  | executes "JMP addr" if ZERO flag isn't set
0x19   | JG   addr  | executes "JMP addr" if GREATER flag is set
0x1A   | JNG  addr  | executes "JMP addr" if GREATER flag isn't set
0x1B   | JL   addr  | executes "JMP addr" if LOWER flag is set
0x1C   | JNL  addr  | executes "JMP addr" if LOWER flag isn't set
0x1D   | JC   addr  | executes "JMP addr" if CARRY flag is set
0x1E   | JNC  addr  | executes "JMP addr" if CARRY flag isn't set
0x1F   | LOOP addr  | executes "JMP addr" and "DEC LC" if LC > 0
-------+------------+-----------------------------------------------------------

--- ALU flags ------------------------------------------------------------------
flag     | description
---------+----------------------------------------------------------------------
ZERO     | set by CMP if a = b
LOWER    | set by CMP if a < b
GREATER  | set by CMP if a > b
OVERFLOW | set by ADD, SUB, MUL, INC and DEC
         |   if 31-bits limit is exceeded (signed carry)
CARRY    | set by ADD, SUB, SAL, SAR, INC, DEC, SHL and SHR
         |   if 32-bits limit is exceeded (unsigned carry)
---------+----------------------------------------------------------------------

--- Values table ---------------------------------------------------------------
value       | description
------------+-------------------------------------------------------------------
0x00 - 0x0F | register (R0 .. R15)
0x10        | loop counter
0x11        | stack pointer
0x12        | program counter
0x80 - 0x8F | [register]
0x90        | [loop counter]
0x91        | [stack pointer]
0x92        | [program counter]
------------+-------------------------------------------------------------------


=== Startup ====================================================================

All registers are set to 0x00000000 when the CPU starts.
